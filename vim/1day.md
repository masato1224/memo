### 2 vimの起動と終了

| key | 意味           |
| --- | -------------- |
| :q! | 保存せずに終了 |

### 3 テキスト編集 - 削除

| key | 意味                 |
| --- | -------------------- |
| x   | 一文字削除（delete） |

### 4 テキスト編集 挿入

| key | 意味                         |
| --- | ---------------------------- |
| i   | 挿入モード(カーソル位置から) |
| I   | 挿入モード(行頭から)         |

### 5 テキスト編集 追加

| key | 意味                         |
| --- | ---------------------------- |
| a   | 追加モード(カーソル位置から) |
| A   | 追加モード(行末から)         |

### 6 ファイル編集

| key | 意味                     |
| --- | ------------------------ |
| :wq | ファイルを保存して閉じる |
